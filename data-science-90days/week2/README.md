---

## ğŸ› Common Beginner Mistakes

1. **Forgetting to import pandas**: Always start with `import pandas as pd`
2. **Modifying original data**: Use `.copy()` when cleaning
3. **Using `and`/`or` instead of `&`/`|`** in filters
4. **Not resetting index** after dropping rows
5. **Ignoring missing data** - always check with `.isnull().sum()`
